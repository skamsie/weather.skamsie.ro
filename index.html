INFO

INPUT: Type in the input box the location for which you want to get weather information. For big cities providing only the city name usually suffices, but for less known locations it is better to provide also the zip code, country name, or country code.

UNITS: Toggling the 'units button' to °F will also switch to the imperial system for the wind speed information. The units are changed only after submitting.

GEOLOCATION: Click to show weather information for your current location. The button is hidden on browsers that do not support geolocation.
